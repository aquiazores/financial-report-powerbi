## Financial Report Dashboard (Power BI)
**Tool:** Power BI  
**Dataset:** Superstore Sales (Orders)  
**Pages:** Financial Report | Product Deep Dive  

### Key Insights
- Technology has the highest profit margin at 17.40%
- Furniture underperforms at only 2.49% margin despite high sales
- Tables and Bookcases are loss-generating sub-categories
- Heavy discounting (30%+) negatively correlates with profit
